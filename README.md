# Evernight 한국어 패치 리소스팩

Minecraft: Java Edition 1.18.2 모험 맵 [Evernight](https://www.planetminecraft.com/project/evernight-5231097/)의 한국어 리소스팩입니다.

## 적용 방법

1. [최신 릴리스](https://github.com/legenlee/evernight-korean/releases/) 페이지에서 Evernight-Korean.zip을 다운받은 후, 마인크래프트 리소스팩 메뉴에 추가하여 최상단으로 적용합니다.
2. 마인크래프트 언어를 **한국어 (대한민국)**로 변경합니다.
3. Evernight 맵을 열어 텍스트가 한국어로 표시되는지 확인합니다.
